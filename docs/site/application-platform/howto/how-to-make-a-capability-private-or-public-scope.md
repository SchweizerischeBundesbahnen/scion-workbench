## [SCION Workbench Application Platform][menu-home] > [How To Guides][menu-how-to] > Capability

#### How to make a capability private or public scope

In the manifest, go to the capability and set `private` property to either `true` or `false`. If private (or if not specified), other applications cannot use this capability.
